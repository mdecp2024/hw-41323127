第一題:01_string_i...ut_print.zip

說明:學習如何處理和輸出字符串,還有如何處理input(),print().最後學習如何創建簡單的 ASCII 藝術。

第二題:02_variables.zip

說明:這段程式碼的目的是讓你了解如何與用戶互動，並將他們的輸入與其他文本合併輸出。   

第三題:03_list_if_in_else.zip

說明:

這段程式碼模擬了一個書店的簡單查詢系統，顧客可以選擇想要購買的程式設計書籍，然後系統會檢查書店是否有這本書。如果有，就告訴顧客書店有賣；如果沒有，就告訴顧客書店沒有這本書。

第四題:04_list_ap..._remove.zip

說明:學習

  • 使用 input() 向自己詢問並更新購物清單。
  • 使用 append() 方法將新物品添加到清單中。
  • 使用 remove() 方法從清單中移除物品。

    • 第五題:05_list_ind...p_insert.zip

    說明:學習

  • 使用 .index() 查找項目在列表中的位置。
  • 使用 .pop() 移除列表中的項目。
  • 使用 .insert() 在列表的特定位置插入新項目。

    • 第六題:06_list_slicing.zip

    說明:學習

    • 使用正索引和負索引來選擇特定的項目。
    • 使用步長來選擇每隔一個的項目。
    • 可以對列表進行反向排序。

    第七題:07_list_slicing_use.zip

    說明:

    • 使用索引來修改列表元素。
    • 使用切片來替換列表中的一部分。
    • 使用 del 關鍵字來刪除列表中的元素或整個列表。

    第八題:08_for_range.zip

    說明:這段程式碼展示了如何使用 for 迴圈遍歷列表並將不同列表的元素對應起來顯示。

    第九題:09_for_loo...f_equals.zip

    說明:

    • 程式遍歷 animals 列表並檢查每個動物是否是你想看的動物。
    • 如果是你想看的動物，會輸出特別的訊息；否則，會輸出普通的訊息。
    • 這段程式碼展示了如何使用 for 迴圈和 if 條件來處理列表中的元素。
    第十題:10_for_search.zip
    說明:
    • 這段程式碼展示了如何根據配件名稱的長度或位置篩選並輸出特定的配件。
    • 使用 for 迴圈遍歷配件列表，配合 if 條件語句，根據長度和位置條件顯示配件及其索引。
